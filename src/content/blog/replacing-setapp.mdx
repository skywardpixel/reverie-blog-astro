上班之后自己的 MacBook 用的越来越少了，也想着节省一下在软件订阅上的开支。SetApp $108 的年费已经交了四年，当时抱着「了解一下 Mac 上好用的软件」的想法现在也已经不太适用了。今天离订阅到期还有十天，于是将装在 MacBook 上的 SetApp 软件清理了一遍，搞清楚哪些软件是真正实用的，并试图找到它的平替。

## 文件管理器 Forklift → Marta

Forklift 属于日常必用的软件了，好像并没有找到开源的替代品。我最终选择了 [Marta](https://marta.sh/)，一款略显简陋但是支持内置终端的 side-by-side 文件管理器。Marta 没有 Forklift 那样的侧边栏，但是支持类似 VSCode 中 Command+P 这样的快速搜索，用久了之后希望能习惯。

## 菜单栏整理 Bartender → Dozer

软件装的多了 Mac 的菜单栏就会异常拥挤。Bartender 可以隐藏部分不常用的菜单栏图标。我找到的免费代替品是 [Dozer](https://github.com/Mortennn/Dozer)，可以说是有过之而无不及。

## 剪贴板管理 Paste → Raycast

我应该算是 Raycast 的轻度用户，之前都不知道这个剪贴板管理的功能。其实我之前就没怎么用到过 Paste，但是偶尔用到的时候就很救命；既然 Raycast 就提供这个功能，那就正好少装一个软件了。

Raycast 另外一个我知道但不太用的功能是窗口管理，比如可以让窗口占据左边三分之一的屏幕。13 寸的屏幕实在太小了，用不上分屏的功能…… 或许可以考虑在公司的电脑上多用用。

## 截屏 Cleanshot X → 系统自带

Cleanshot X 也算是个锦上添花的软件，用了几年 Mac 终于能记住 Command+Shift+3 这个蜜汁快捷键了。

## 批量重命名 Renamer → ？

Renamer 可以用 Regex 批量替换文件名，在把整季度的番剧导入 Plex 的时候经常会用到 (好像术语叫「刮削」)。还没有找到靠谱的代替品，可能会考虑自己写个 Python 脚本来识别不同字幕组的命名模式并自动处理和归类。

## 系统清理 CleanMyMac X → ？

这算是个毁誉参半的软件，有些人觉得这东西和 360 管家差不多。我偶尔会跑一下，一般能至少清理出 50 GB 的空间。打算先手动清理下载文件夹和垃圾箱试试看效果。

## 网盘挂载 CloudMounter → X

之前上学搞 research 的时候常用 CloudMounter 挂载 S3 bucket 来上传数据之类的，使用的心智负担比用 aws cli 稍低一些。现在没有什么使用 S3 的需求了，也就不需要 CloudMounter 了。但是我仍然觉得 CloudMounter 是 SetApp 中最好用的几个软件之一。

总之折腾下来一个月能省下来 10 刀。SetApp 大多数的软件其实都用不上，也就以上几个算是实用的，即使是直接找软件提供商买断应该也比每个月花 10 刀划算些。对于一台平时也不用来干活的电脑来说就更加不值了。好在光靠 Raycast 就能解决不少需求，希望 Raycast 能越来越好。
